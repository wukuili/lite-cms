Ubuntu 20.04桌面版的系统还原主要针对系统文件、配置或出厂状态的恢复，以下是两种常用方法，适用于不同场景：


### **方法一：通过Timeshift（时间点快照还原，推荐）**
Timeshift是Ubuntu官方推荐的系统快照工具，类似Windows的“系统还原点”，可备份系统文件、配置并恢复到之前的状态（需提前创建快照）。

#### **适用场景**
- 系统误操作、安装软件导致崩溃，但用户数据（如`/home`目录文件）需保留。
- 已提前通过Timeshift创建过系统快照。

#### **操作步骤**
1. **安装Timeshift（若未安装）**  
   打开终端（`Ctrl+Alt+T`），执行命令：  
   ```bash
   sudo apt update && sudo apt install timeshift
   ```

2. **创建快照（首次使用需先做）**  
   - 启动Timeshift（在应用菜单搜索“Timeshift”），首次打开选择快照类型（推荐“RSYNC”，兼容多数文件系统）。  
   - 选择快照存储位置（建议非系统盘，如外接硬盘），点击“Create”创建初始快照（命名格式如`2024-11-19_10:00:00`）。

3. **执行还原**  
   - 启动Timeshift，在左侧“Snapshot”列表中选择需要恢复的快照（如崩溃前的快照），点击“Restore”。  
   - 按照向导选择目标分区（通常是系统盘`/`所在分区），确认后点击“Next”，系统会自动重启并完成还原。


### **方法二：通过重新安装系统（彻底还原，适合无快照场景）**
若未创建快照，或系统损坏严重（如无法启动），可通过重新安装Ubuntu 20.04实现“还原”，保留用户数据（需手动备份）。

#### **适用场景**
- 无快照备份，系统无法修复（如内核损坏、引导故障）。
- 希望恢复到系统初始状态（类似“重装系统”）。

#### **操作步骤**
1. **备份用户数据**  
   - 若系统可启动：将`/home/用户名`下的重要文件（文档、图片等）复制到U盘或移动硬盘。  
   - 若系统无法启动：通过Ubuntu安装U盘启动进入“试用模式”，挂载原系统分区，复制`/home`目录数据到外部存储。

2. **制作安装U盘**  
   - 下载Ubuntu 20.04.3桌面版ISO镜像（[官网链接](https://releases.ubuntu.com/20.04/)）。  
   - 使用工具（如Rufus、Etcher）将ISO写入U盘，制作启动盘。

3. **重新安装并保留数据**  
   - 从U盘启动电脑，选择“Install Ubuntu”。  
   - 安装过程中，在“安装类型”步骤选择“清除整个磁盘并安装Ubuntu”（彻底还原），或选择“与其他系统共存”并格式化系统分区（保留其他分区数据）。  
   - 完成安装后，将备份的`/home`数据复制回新系统即可。


### **注意事项**
- Timeshift默认不备份`/home`目录（仅备份系统文件），若需备份用户数据，可手动勾选或使用`rsync`单独备份。  
- 重新安装系统前，务必确认数据已备份，避免丢失。  
- 若系统引导损坏，可先尝试通过`boot-repair`工具修复（需从安装盘启动后安装该工具），无需直接重装。

根据你的场景选择合适的方法，有具体步骤疑问可进一步说明。
