做亚马逊产品推荐博客，SEO是核心命脉——**没有流量，就没有点击和佣金**。2025年，Google算法更强调 **E-E-A-T（经验、专长、权威、信任）**、**用户体验（Core Web Vitals）** 和 **内容真实性**（打击AI泛滥文）。下面是**从0到1的实战SEO体系**，专为联盟营销博客优化，兼顾Hugo（静态）和WordPress（动态）实现。

---

## 一、核心策略：三大支柱

| 支柱 | 目标 | 2025重点 |
|------|------|---------|
| **内容质量** | 真实评测 > AI堆砌 | 必须有**个人使用照片/视频** |
| **技术SEO** | 速度+移动优先 | LCP < 2.5s，CLS < 0.1 |
| **外链与品牌** | 信任信号 | 获得Reddit/X/知乎引用 |

---

## 二、关键词研究（选品+内容双驱动）

### 1. 工具组合（免费为主）
| 工具 | 用途 | 推荐指数 |
|------|------|---------|
| **Google Keyword Planner** | 搜索量/竞争度 | ⭐⭐⭐⭐ |
| **Ubersuggest**（免费版） | 长尾词挖掘 | ⭐⭐⭐ |
| **AnswerThePublic** | 问答类关键词 | ⭐⭐⭐ |
| **Amazon搜索框** | 买家真实语言 | ⭐⭐⭐⭐ |

### 2. 关键词类型（联盟博客专属）
| 类型 | 示例 | 策略 |
|------|------|------|
| **商业意图** | “best wireless earbuds 2025” | 主攻，转化高 |
| **评测意图** | “AirPods Pro 2 vs Sony WF-1000XM5” | 深度对比文 |
| **问题意图** | “Kindle Paperwhite值得买吗” | 问答式结构 |
| **长尾词** | “适合程序员的机械键盘推荐 1000元内” | 竞争低，易上位 |

> **技巧**：用亚马逊搜索“关键词+字母”（如“耳机 a”），看自动补全，挖掘真实买家语言。

## 三、内容创作：E-E-A-T是命根子

### 必须包含的“信任元素”
| 元素 | 如何实现 | 举例 |
|------|---------|------|
| **个人体验** | 上传**真实开箱照/使用视频** | “我用了一个月，续航实测8小时” |
| **数据支撑** | 电池测试、降噪分贝 | 用手机App测噪音 |
| **更新机制** | 每年Q1更新“2025版” | 旧文加`<ins datetime="2025-01">` |
| **作者简介** | 博客加`rel="author"` | “写过50篇耳机评测” |

### 推荐文章结构（模板化）
```markdown
# [产品]值得买吗？2025真实评测

## 我为什么买它（痛点引入）
## 开箱&第一印象（照片）
## 实测表现（数据+场景）
## 优缺点对比表
## 谁适合买？谁别买？
## 购买链接（联盟链接）
```

## 四、技术SEO（Hugo vs WordPress）

| 项目 | Hugo实现 | WordPress实现 |
|------|---------|--------------|
| **速度优化** | 静态生成，天生快 | 用LiteSpeed + QUIC.cloud |
| **结构化数据** | `layout`中加JSON-LD | Yoast/RankMath自动生成 |
| **移动适配** | Tailwind响应式 | Elementor移动预览 |
| **图片优化** | Hugo pipe + WebP | Smush插件 |

### 必装结构化数据（Schema）
```json
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "AirPods Pro 2",
  "review": {
    "@type": "Review",
    "author": {"@type": "Person", "name": "你的名字"},
    "reviewRating": {"@type": "Rating", "ratingValue": "4.5"}
  },
  "offers": {
    "@type": "Offer",
    "url": "你的联盟链接",
    "priceCurrency": "CNY",
    "price": "1899"
  }
}
</script>
```

## 五、站内优化（让Google爱上你）

### 1. URL结构
```
yoursite.com/best-wireless-earbuds-2025
```
- 短、含关键词、无动态参数

### 2. 内链策略
- **枢纽页**：创建“耳机推荐”分类页，链接所有评测
- **相关推荐**：每篇文末加“你可能还喜欢”
- **面包屑导航**：`首页 > 耳机 > AirPods Pro 2`

### 3. 图片ALT文本
```html
<img src="airpods-unboxing.jpg" alt="AirPods Pro 2开箱实拍 - 充电仓细节">
```

## 七、追踪与迭代（数据驱动）

| 工具 | 功能 |
|------|------|
| **Google Analytics 4** | 流量来源、跳出率 |
| **Google Search Console** | 关键词排名、点击率 |
| **热图工具（Hotjar免费版）** | 用户点击联盟链接位置 |

### 关键指标
- **点击率（CTR）** > 2% → 优化标题
- **平均停留时间** > 2分钟 → 内容够深度
- **联盟转化率** > 1.5% → 选品正确

## 八、2025年避坑清单

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| AI全文生成 | Helpful Content Update降权 | AI打草稿，手动重写+照片 |
| 堆砌联盟链接 | 用户反感+SEO惩罚 | 每篇≤3个，自然融入 |
| 不更新旧文 | 排名下滑 | 每年更新价格/新款 |
| 忽略移动端 | Core Web Vitals红灯 | 优先手机预览 |
